# OpenAPI Versioning Examples

This repository contains examples of API versioning strategies in ASP.NET Core with OpenAPI documentation.

## Projects

### Minimal API

- **queryheader-versioning-aot**: Query string and header-based API versioning with Native AOT compilation (.NET 10 RC2)

## Purpose

These projects demonstrate different approaches to API versioning and their compatibility with various .NET features like Native AOT compilation.
